# Storage Concepts:

## Storage

Storage is where data is saved so it can be used now or later. It
ensures durability and availability of data.

------------------------------------------------------------------------

## Backup vs Archive

### Backup

-   Used for quick recovery from failures
-   Taken frequently (daily/hourly)
-   Fast restore time
-   Example: Restoring a crashed database

### Archive

-   Used for long-term storage
-   Rarely accessed
-   Slow retrieval but low cost
-   Example: Old logs for compliance

------------------------------------------------------------------------

## HDD vs SSD

### HDD (Hard Disk Drive)

-   Mechanical disk
-   Slower performance
-   Uses **Throughput** (MB/s)
-   Best for large files, backups

### SSD (Solid State Drive)

-   No moving parts
-   Very fast
-   Uses **IOPS** (operations/sec)
-   Best for databases, OS

------------------------------------------------------------------------

## Performance Metrics

### IOPS

-   Number of read/write operations per second
-   Used for small, random data access

### Throughput

-   Amount of data transferred per second
-   Used for large, sequential data

------------------------------------------------------------------------

## Storage Units

### Decimal (Base 1000)

-   KB = 1000 Bytes
-   MB = 1000 KB
-   GB = 1000 MB
-   TB = 1000 GB

### Binary (Base 1024)

-   KiB = 1024 Bytes
-   MiB = 1024 KiB
-   GiB = 1024 MiB
-   TiB = 1024 GiB

------------------------------------------------------------------------

## Key Differences

  Concept      Meaning
  ------------ ----------------------
  Backup       Fast recovery
  Archive      Long-term storage
  HDD          Cheap, slower
  SSD          Fast, expensive
  IOPS         Number of operations
  Throughput   Data transfer size

------------------------------------------------------------------------

## Real-World Usage

-   Databases → SSD (High IOPS)
-   Video streaming → HDD (High Throughput)
-   Backup → Object Storage
-   Archive → Cold Storage

------------------------------------------------------------------------

## Summary

-   Backup = Quick restore
-   Archive = Long-term storage
-   SSD = Speed
-   HDD = Capacity
-   IOPS = Operations
-   Throughput = Data size
